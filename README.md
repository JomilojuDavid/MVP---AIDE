# AIDE - From Stuck to Clear & Confident

## About

AIDE is your personalized journey where mindset mastery meets business growth. This application helps guide users through a transformative experience with interactive quizzes, resources, and personalized dashboards.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```
